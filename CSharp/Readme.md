# C# 문제풀이


## Compile & Execute
----
csc 컴파일러를 이용한 단일 .cs 파일 컴파일
```
csc File.cs
```

tempalate-generator를 통해 생성된 파일 컴파일 시
```
csc -define:TEST File.cs
```
